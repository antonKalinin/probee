## Probee Tutor

App becomes a tutor for user and helps in learning new things/improving skills.
Core idea is that app can prepare the task by observing user activity and then show the task at given schedule.
Task could accept input in different modalities: text, voice. For example if user wants to learn new language,
app can provide short grammar notices from time to time during the day and later ask to translate phrase or word.
Same could be done for math or programming. Biggest difficulty is make app not noisy or annoying. That it why it should be'
more like a game, where user do not know the topic of the task but it should be very relevant to user activity.
